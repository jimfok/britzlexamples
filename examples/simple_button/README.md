# Simple Button
This example shows how to create a button using a gui scene with an optional debug log panel.

## Features
- Two interactive buttons with visual feedback ("CLICK ME!" and "NO, CLICK ME!")
- Debug log panel (toggled by clicking the "DEBUG" button in top-right)
- Large debug panel covering 90% of screen in bottom-left area
- Real-time logging of all button interactions with timestamps
- Console output for debugging

## Controls
- **CLICK ME!** and **NO, CLICK ME!**: Regular buttons that log interactions
- **DEBUG**: Toggle button to show/hide the debug log panel
